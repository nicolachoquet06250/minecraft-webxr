# Items disponibles

Cette page liste les items actuellement disponibles dans le jeu.

## Fonctionnement général

Dans l'état actuel du jeu :

- tous les blocs définis sauf `Air` ont un item associé ;
- les items de blocs ont une taille maximale de stack de 64 ;
- certains outils ou items spéciaux peuvent avoir une taille de stack différente ;
- les items peuvent apparaître dans l'inventaire, le craft et les drops.

## Items de blocs

Tous les blocs listés dans [Blocs disponibles et où les trouver](./blocks.md) existent aussi comme items, sauf `Air`.

Exemples :

| Item | Nom français | Stack max | Comment l'obtenir |
|------|--------------|-----------|-------------------|
| Grass Block | bloc d'herbe | 64 | Casser un bloc d'herbe. |
| Dirt | terre | 64 | Casser de la terre. |
| Stone | pierre | 64 | Casser de la pierre. |
| Sand | sable | 64 | Casser du sable. |
| Water | eau | 64 | Item défini, pas forcément accessible en survie normale. |
| Oak Log | bûche de chêne | 64 | Casser un tronc de chêne. |
| Oak Leaves | feuilles de chêne | 64 | Casser des feuilles ou attendre leur chute automatique. |
| Poppy | coquelicot | 64 | Casser un coquelicot généré dans les plaines. |
| Oak Planks | planches de chêne | 64 | Craft avec une bûche de chêne. |
| Crafting Table | table de craft | 64 | Craft avec 4 planches de chêne. |

## Item spécial : pioche en terre et herbe

| Item | Nom français proposé | Stack max | Comment l'obtenir |
|------|----------------------|-----------|-------------------|
| Dirt Grass Pickaxe | pioche en terre et herbe | 1 | Craft avec 3 blocs d'herbe et 2 blocs de terre. |

La pioche en terre et herbe utilise une image dédiée :

```txt
/items/pickaxe-grass-dirt.png
```

## Item spécial : poppy

Le coquelicot utilise une image dédiée dans l'inventaire :

```txt
/blocks/plants/poppy.png
```

Dans le monde et en drop, la poppy peut utiliser un modèle 3D dédié.

## Remarques

- Un item défini n'est pas forcément obtenable naturellement pour l'instant.
- Les items de blocs peuvent généralement être placés dans le monde si le joueur les possède dans son inventaire.
- Les stacks de blocs montent à 64.
- La pioche en terre et herbe est limitée à 1 par stack.
