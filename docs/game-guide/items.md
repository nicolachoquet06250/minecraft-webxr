# Items disponibles

Cette page liste les items actuellement disponibles dans le jeu.

## Fonctionnement general

Tous les blocs definis sauf `Air` ont un item associe. Les items de blocs ont une taille maximale de stack de 64. Les outils peuvent avoir une limite differente.

## Items de blocs

Tous les blocs listes dans [Blocs disponibles et ou les trouver](./blocks.md) existent aussi comme items, sauf `Air`.

## Item special

- Dirt Grass Pickaxe / pioche en terre et herbe : stack max 1, craft avec 3 blocs d'herbe et 2 blocs de terre.
- Poppy / coquelicot : utilise une image dediee dans l'inventaire.

- Minerais

- Plantes
