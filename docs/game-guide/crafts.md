# Crafts possibles

Cette page liste toutes les recettes de craft actuellement implémentées.

## Lire les patterns

Les recettes utilisent une grille 3x3.

Dans les schémas :

- `.` signifie case vide ;
- chaque autre symbole représente un ingrédient ;
- le résultat est indiqué sous le schéma.

## Planches de bois

Les recettes de planches fonctionnent avec une seule bûche placée dans n'importe quelle case de la grille.

### Oak Planks / planches de chêne

Ingrédient :

- `O` = Oak Log / bûche de chêne

```txt
O . .
. . .
. . .
```

Résultat :

```txt
4x Oak Planks / planches de chêne
```

La bûche peut être placée dans n'importe quelle case de la grille.

## Table de craft

### Crafting Table / table de craft

Ingrédient :

- `P` = Oak Planks / planches de chêne

```txt
P P .
P P .
. . .
```

Résultat :

```txt
1x Crafting Table / table de craft
```

Remarque : la recette actuelle utilise uniquement des planches de chêne.

## Outils

### Dirt Grass Pickaxe / pioche en terre et herbe

Ingrédients :

- `G` = Grass Block / bloc d'herbe
- `D` = Dirt / terre

```txt
G G G
. D .
. D .
```

Résultat :

```txt
1x Dirt Grass Pickaxe / pioche en terre et herbe
```

## Remarques

- Les recettes de planches acceptent la bûche dans n'importe quelle case de la grille.
- La table de craft actuelle ne se craft qu'avec des planches de chêne.
- Les autres types de planches ne créent pas encore de table de craft.
- Aucun craft de four, coffre, torche, laine ou blocs décoratifs n'est encore défini.
