## Remarques sur l'obtention

- Les blocs générés naturellement peuvent être obtenus en les cassant.
- Les feuilles peuvent aussi tomber automatiquement après la destruction complète d'un arbre.
- Les blocs craftables peuvent être obtenus via l'interface de craft.
- Les blocs définis mais non générés naturellement existent déjà côté jeu, mais ne sont pas forcément accessibles en survie normale pour l'instant.
